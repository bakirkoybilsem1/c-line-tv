# C-LINE Studio

**A Common Language for Integrative Entrepreneurship (C-LINE)**

C-LINE Studio is an interactive, student-centered platform designed to support visual communication, sign-based learning, and integrative thinking through collaborative content creation.

The platform enables students to contribute by uploading **photos and videos** that represent concepts, while educators maintain quality through a structured **approval system**.

---

## 🌍 Features

- 📷 Photo-based content creation  
- 🎥 Video-supported expression and learning  
- 👨‍🏫 Teacher approval & moderation system  
- 🌐 Multilingual interface  
  - Turkish (TR)  
  - English (EN)  
  - Latvian (LV)  
  - Italian (IT)  
- 🎮 Interactive and visual learning experience  

---

## 👩‍🎓 How It Works

1. Students log in to the platform  
2. They submit a concept or word  
3. They upload photo and video representations  
4. Teachers review and approve submissions  
5. Approved content becomes part of the shared system  

---

## 🧩 System Architecture

- **Frontend:** Next.js (Vercel)
- **Backend:** Supabase (Database + Auth + Storage)
- **Deployment:** Vercel
- **Version Control:** GitHub

## 🎯 Purpose

C-LINE Studio aims to:

- Support visual and experiential learning  
- Encourage student-generated content
